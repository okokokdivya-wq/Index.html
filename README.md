# Digital Marketing Proposal — Bag Brand

A GitHub Pages-ready professional proposal website for a 3-month digital marketing engagement.

## 📋 Overview

This is a modern, responsive proposal webpage designed by **Divya & Punam** for a Bag Brand client in Chennai. The site showcases a complete 3-month digital marketing engagement including:

- **Phase 1:** Content & Instagram Management
- **Phase 2:** Amazon Setup & Management
- **Phase 3:** Meta Ads Management

## 🎨 Design Features

- **Modern Typography** — Playfair Display (serif), Inter (sans-serif), JetBrains Mono (monospace)
- **Elegant Color Palette** — Dark ink, gold accents, cream backgrounds
- **Responsive Layout** — Fully mobile-friendly
- **Smooth Animations** — Scroll-triggered fade-in effects
- **Print-Friendly** — Optimized for PDF export

## 🚀 Getting Started

### Viewing Locally

1. Clone the repository:
   ```bash
   git clone https://github.com/okokokdivya-wq/marketing-proposal.git
   cd marketing-proposal
   ```

2. Open `index.html` in your browser

### Deploying to GitHub Pages

1. Go to your repository settings
2. Under "Pages," select "Deploy from a branch"
3. Choose the `main` branch as the source
4. Save — your site will be live at `https://okokokdivya-wq.github.io/marketing-proposal/`

## 💻 Customization

### Updating Content

Edit `index.html` to customize:
- Client name and location
- Team names
- Budget amounts
- Services and deliverables
- Timeline and content strategy

### Changing Colors

Modify the CSS variables in the `<style>` section:
```css
:root {
  --ink:      #0D1117;
  --gold:     #C8963A;
  --cream:    #FAFAF7;
  /* ... etc ... */
}
```
